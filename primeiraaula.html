Adobe DreamWeaver 

O que é e para que serve o DreamWeaver? 

O Adobe Dreamweaver é um programa de desenvolvimento web que permite criar, editar e gerenciar sites e aplicações web. Este é o conteúdo da tag Layout P Ele é uma ferramenta útil para web designers, desde iniciantes até profissionais avançados, permite criar, editar e gerenciar sites e aplicações web.

 Como trabalhamos ou fazemos?

Usamos de 2 forma: criamos dites por códigos html e linguagem de programação, ou podemos criar conteúdos dos sites incorporando os elementos por item no menu.
